# Instruções para Ícones do Vale das Minas

Coloque os seguintes arquivos nesta pasta:

1. **icon.png** (1024x1024 pixels)
   - Ícone principal do app
   - Formato PNG com fundo transparente
   - Resolução mínima: 1024x1024 pixels

2. **adaptive_icon.png** (432x432 pixels)
   - Parte do ícone que ficará em primeiro plano no Android
   - Apenas a parte central do ícone (sem fundo)
   - Formato PNG com fundo transparente

## Cores do Tema Vale das Minas:
- Cor primária: #4CAF50 (verde)
- Cor secundária: #FFC107 (amarelo)

## Dicas de Design:
- Use as cores do tema do cliente
- Mantenha o design simples e reconhecível
- Teste em diferentes tamanhos e fundos
