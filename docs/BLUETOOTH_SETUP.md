# Configuração Bluetooth para Impressoras Térmicas

Este documento descreve como configurar o Bluetooth para permitir conexão e impressão em impressoras térmicas no aplicativo.

## Pacote Utilizado

O aplicativo usa o pacote **`flutter_reactive_ble`** (versão 5.3.1+) que é:
- ✅ **100% gratuito** e open source (MIT License)
- ✅ Mantido pela **Philips Hue** (empresa confiável)
- ✅ Suporte completo para iOS, Android e macOS
- ✅ API reativa e moderna com Streams
- ✅ Excelente para dispositivos BLE como impressoras térmicas

```yaml
dependencies:
  flutter_reactive_ble: ^5.3.1
```
